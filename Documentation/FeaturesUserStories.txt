User Stories
As a user, I want to be able to feed my virtual pet to keep it alive.
As a user, I want to be able to satisfy my pet. 
As a user, I want to feel like my pet loves me.
As a user, I want to be able to choose between different types of animal as my pet.
As a user, I want my pet to possess different attributes depending on what type of animal my pet is.
As a user, I want to see my pet grow and evolve.
As a user, I want to be able to share my adventures with friends so everyone can have fun.
As a user, I want to teach my pet things so I can feel progress in the game.
